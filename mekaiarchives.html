Mek-Ai: Archives places players inside a fractured corporate system — a Meka Corporation laptop left running long after it should have gone dark. Guided by Ava Keep, players navigate corrupted file structures, recover missing employee records, and decrypt communications that were never meant to be read.

The deeper the investigation goes, the more the system resists. A corporate Fixer is monitoring the breach in real time. Every file opened is a trade — information for exposure.

02 — The System

Layers of
access. Layers
of truth.

The Meka Corporation archive is structured across clearance tiers. Each tier unlocks new files, new employee records, and new evidence of what the corporation was actually running beneath its public-facing operations. Progress is earned through observation and puzzle-solving — not brute force.

Mek-Ai: Archives is a companion experience to HourGlass: Missing Time. Events in one inform the other. The full picture only emerges when both are explored.
